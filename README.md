Este repositório serve como a vitrine do meu portfólio profissional, reunindo aplicações web premium, fluidas e com alta sensibilidade estética. ## ⚡ A Base Operacional: `_Default-Project`

Para manter a consistência arquitetônica e acelerar o desenvolvimento de novas ideias, este repositório possui uma base de partida padronizada na pasta [\_Default-Project](file:///e:/Git/SiteProjects/_Default-Project):

* **O que é?**: Um boilerplate minimalista e ultraotimizado configurado para servir de base para novos desenvolvimentos no ecossistema.
* **Stack Integrada**:
  * **React 19 & React Router 7**: A vanguarda do desenvolvimento reativo e roteamento de arquivos client-side.
  * **Vite 8**: Ferramental e empacotador de altíssima velocidade para carregamento instantâneo (HMR).
  * **Tailwind CSS v4**: Integração nativa e compilada direto no Vite para estilizações rápidas e consistentes.
  * **Linters & Formatadores**: ESLint e Prettier pré-configurados com regras rígidas de formatação e boas práticas.
  * **Utilitários de Estilo**: Setup prévio de `clsx` e `tailwind-merge` para manipulação dinâmica e segura de classes CSS.

## 🛠️ Como Clonar e Executar um Projeto

1. Certifique-se de ter o [Node.js](https://nodejs.org/) instalado.
2. Navegue até a pasta do projeto desejado (ex: `compativeis`):
   ```bash
   cd compativeis
   ```
3. Instale as dependências configuradas:
   ```bash
   npm install
   ```
4. Rode o ambiente de desenvolvimento local:
   ```bash
   npm run dev
   ```

Para criar um novo projeto a partir da nossa fundação, basta duplicar a pasta `_Default-Project`, renomeá-la e rodar o comando de instalação.
